Особистий сейф PWA v4 Secure

Додано:
- PIN при першому запуску
- AES-GCM шифрування всієї бази
- PBKDF2-SHA256 для отримання ключа з PIN
- Зашифроване локальне зберігання
- Зашифрований експорт .vault
- Імпорт .vault
- Автозамок: 1/5/15/30 хв або вимкнено
- Блокування після 5 неправильних PIN на 60 секунд
- Privacy overlay при згортанні додатку
- Опційний Passkey/WebAuthn для Face ID/Touch ID, якщо браузер і HTTPS дозволяють

Важливо:
- PIN не відновлюється. Якщо забути PIN, зашифровану базу не розшифрувати.
- Face ID у PWA працює через Passkey/WebAuthn. Він не замінює PIN як ключ шифрування.
- Для реального релізу краще перейти з localStorage на IndexedDB, бо фото/відео швидко збільшують обсяг даних.
