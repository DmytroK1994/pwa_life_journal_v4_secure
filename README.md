# Opus PWA

Opus — особистий цифровий сейф та база знань.

## Що є в цій збірці

- PWA: `manifest.json` + `service-worker.js`
- Mobile First інтерфейс під iPhone
- Реєстрація користувача
- Вхід через логін і пароль
- PIN 4 або 6 цифр
- Face ID повністю прибрано
- IndexedDB для записів і вкладень
- AES-GCM шифрування тексту та файлів
- PBKDF2 SHA-256 для ключів
- Фото, відео, аудіо, документи
- Голосові нотатки через мікрофон
- Експорт `.vault`
- Експорт відкритого JSON після розблокування
- Офлайн-робота

## Запуск

Просто відкрий `index.html` локально або залий файли в GitHub Pages.

Для GitHub Pages:
1. Створи репозиторій.
2. Завантаж усі файли з архіву в корінь репозиторію.
3. Settings → Pages → Deploy from branch → main → root.
4. Відкрий посилання GitHub Pages.

## Важливо

Це локальний PWA без сервера. Логін/пароль працюють для локального сейфа в браузері.
Для справжньої синхронізації між пристроями потрібен бекенд: Firebase, Supabase або власний сервер.
